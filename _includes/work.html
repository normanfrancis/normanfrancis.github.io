Ironside Computers 🕹️

Woocommerce - JS - CSS

Ironside is a site that allows users to build custom gaming PCs. I extended the product customizer to allow for organizing components into tabs and collapsible sections. Individual components were further extended to present additional information, such as specs and videos in modals.

Seals 🔒

Magento - JS - CSS

As the name implies Seals provides industrial seals - not the animal - for various applications. The product page was extended to allow for conditional custom options that changed based on what the user selected. The quantity selector was extended to allow users to select quantities from a dropdown based on the product’s tier pricing in the backend.

Brave Gunner 🚀

ActionScript - Wiicade API - Design

Brave Gunner is 2D vertical shmup inspired by some of my favorites video games. I drew the designs which then made into assets by an artist. The sound effects were made using various VSTs. A novel feature at the time was that the game utilized the Wiicade API to allow players to use the Wii remote to play when using the Wii’s browser.
